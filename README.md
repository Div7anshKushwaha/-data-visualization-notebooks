# 📊 Data Visualization with Python

A comprehensive collection of Jupyter notebooks covering the most popular Python data visualization libraries — Matplotlib, Seaborn, Plotly Express, Plotly GO, and Plotly Dash.

## 📁 Notebooks Overview

| Notebook | Library | Description |
|----------|---------|-------------|
| `Matplotlib 1.ipynb` | Matplotlib | Core plotting — line, bar, scatter, histograms, subplots |
| `Matplotlib 2.ipynb` | Matplotlib | Advanced customization — styles, annotations, twin axes |
| `Seaborn 1.ipynb` | Seaborn | Statistical plots — heatmaps, pairplots, boxplots, violin |
| `Seaborn 2.ipynb` | Seaborn | Advanced Seaborn — FacetGrid, regression plots, themes |
| `plotly_express.ipynb` | Plotly Express | Interactive charts with minimal code — px API |
| `Plotly_go.ipynb` | Plotly GO | Low-level interactive plots using graph_objects |
| `Plotly_dash.ipynb` | Plotly Dash | Building interactive web dashboards with Python |

## 🛠️ Libraries Used

- **Matplotlib** — foundational static plotting library
- **Seaborn** — statistical visualization built on Matplotlib
- **Plotly Express** — fast, interactive charts with one-liners
- **Plotly Graph Objects** — full control over interactive figures
- **Plotly Dash** — web-based dashboards powered by Python

## 🚀 Getting Started

```bash
pip install matplotlib seaborn plotly dash
```

Open any `.ipynb` file in Jupyter or VS Code and run the cells.
